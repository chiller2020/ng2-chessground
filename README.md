#chessground-ng2wrap

Angular 2 wrapper for https://github.com/ornicar/chessground

Build environment is based on https://github.com/CaptainCodeman/angular2-webpack-polyfill

Work in early progress.

## Setup

    npm install
    
   Run dev server

    npm start

Build deployment

    npm run build
